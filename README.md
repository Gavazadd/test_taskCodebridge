# test_taskCodebridge

### Запуск проекту
- Налаштуйте SQl базу даних, в моєму випадку я використовував Postgres
- Створіть файли .env та вкажіть інформацію:<br>
  PORT = 5000 <br>
  DB_NAME = Назва БД<br>
  DB_USER = Ім'я користувача БД<br>
  DB_PASSWORD = Пароль БД<br>
  DB_HOST = HOST БД<br>
  DB_PORT = PORT БД<br>
- npm ci
- npm run dev
